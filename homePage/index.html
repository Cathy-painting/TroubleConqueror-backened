错题征服者

中小学生错题管理系统 · 让学习更高效，让错题不再困扰

✨ 产品简介

错题征服者是一套专为中小学生设计的错题管理系统，旨在帮助用户高效管理错题、提升学习效率。 通过智能OCR识别、知识图谱构建、个性化练习推荐等功能，让每一道错题都成为进步的阶梯。

🎯 核心功能

🤖

智能错题管理

支持文本输入和拍照识别添加错题，智能分类整理，让错题管理变得简单高效

📸

OCR智能识别

拍照即可自动识别题目内容，支持数学公式识别，解放双手提升录入效率

🏷️

分类标签管理

按学科、年级、难度、知识点进行多维度分类，快速定位薄弱环节

📊

统计分析

错题统计、学习进度分析、知识点掌握情况一目了然，科学评估学习效果

🕸️

知识图谱

构建知识点关联图谱，发现知识薄弱环节，建立完整的知识体系

💪

智能练习

智能推荐练习题目，个性化学习路径，针对性突破薄弱知识点

⚙️ 技术架构

Vue.js Element UI Spring Boot Spring Security Redis Mysql React JWT 若依框架
